Una colección de herramientas en Bash.

## contar_archivos.sh

Cuenta extensiones de archivos en un directorio con múltiples opciones de filtrado y búsqueda recursiva. Permite incluir/excluir extensiones específicas, buscar de forma recursiva, establecer profundidad máxima y más.

## encontrar_duplicados.sh

Encuentra archivos duplicados usando tamaño y hash SHA256. Puede comparar dentro de un solo directorio o entre dos directorios diferentes. Incluye opciones para excluir/incluir patrones, tamaño mínimo y seguimiento de enlaces simbólicos.

## quick_backup.sh

Crea un backup comprimido del directorio actual con marca de tiempo. Ideal para hacer copias de seguridad rápidas de proyectos o directorios de trabajo.

## recortar_imagen.sh

Recorta imágenes eliminando el 20% superior e inferior, guardando los resultados con nombres numerados. Requiere ImageMagick instalado para funcionar correctamente.

## reminder.sh

Crea un temporizador simple que muestra una notificación después de un tiempo especificado (por defecto 540 segundos). Útil para recordatorios rápidos mientras trabajás.

## test_sitio.sh

Verifica la disponibilidad de sitios web usando ping. Incluye soporte para URLs personalizadas y evita duplicados. Ideal para monitorear si tus sitios favoritos están en línea.
